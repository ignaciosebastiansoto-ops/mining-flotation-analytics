# Mining Flotation Process Analytics Dashboard

This project focuses on mining flotation process optimization using Machine Learning and Power BI analytics.

## Project Overview

The objective of this project was to analyze operational variables from a mining flotation plant and develop predictive models capable of estimating iron concentrate performance.

The workflow included:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Random Forest Regression modeling
- Feature Importance analysis
- Interactive Power BI dashboard development

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab
- Power BI

---

## Machine Learning Results

- R² Score: 0.9623
- MAE: 0.0907

The Random Forest model achieved strong predictive performance, demonstrating the potential of machine learning techniques for mining process optimization.
